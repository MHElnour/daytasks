# DayTasks 0.2.0

## Added

- Task cards now show icons for the scheduled date, due date, estimate, and
  priority. Priority appears on the card for the first time.
- Tasks can link more than one project note. Add and remove projects from the
  vault picker in the task editor.
- Redesigned the create and edit task editor into a compact, two-box layout: an
  icon toolbar for status, priority, dates, and estimate at the top, then title
  and description, then tags, contexts, and projects. Status and priority are
  icon menus, and the dates open a picker on click.
- Reserved spots in the editor for subtasks and task dependencies (blocked by /
  blocking), coming in a later release.

## Fixed

- A due date can no longer be set earlier than the task's scheduled date.
