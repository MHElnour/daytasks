# DayTasks 0.1.0

First release.

## Added

- Daily-note task widget rendered at the bottom of `YYYY-MM-DD` daily notes, in
  both Reading and Live Preview modes.
- Create tasks from the widget or the "Create task for current daily note"
  command: title, status, priority, scheduled and due dates, tags, contexts, a
  linked project note, time estimate, and description.
- Edit and delete tasks; cycle status from the card's status pill; completion is
  tracked against a configurable status model.
- Configurable statuses and priorities, plus widget display toggles (task IDs,
  tags, contexts, projects), in the settings tab.
- Tag and project chips: project chips open the linked note, tag chips open the
  global search; both are keyboard-operable.
